# B端页面模块识别补充规则

本文件在 Step1（设计稿模块分析）阶段额外加载，用于引导模型更细粒度地识别 B端页面结构。

## 识别粒度要求

B端页面模块细化规则：
- **筛选区**：单独列为一个模块，描述筛选条件数量、输入框/选择框/时间选择器等组件类型
- **表格区**：列出列数、每列标签名（逐字引用）、是否有操作列、是否有固定列
- **分页器**：单独列出，记录位置（左/右/居中）、格式（如"共 X 条"/"每页 X 条"）
- **操作按钮组**：列出按钮数量和标签（逐字引用），区分主按钮/次按钮
- **Tab / 标签页**：列出每个 Tab 的名称（逐字引用）
- **表单区**：列出字段数量和标签名（逐字引用）
- **弹窗 / 抽屉**：如设计稿中有展示，单独列出

## 输出细化要求

对于表格，content 字段必须包含列名列表，例如：
- content: "表格包含6列：序号、活动名称、状态、创建时间、操作者、操作"

对于筛选区，content 字段必须包含条件列表，例如：
- content: "筛选区包含：活动名称输入框、状态下拉、时间范围选择器、查询按钮、重置按钮"
